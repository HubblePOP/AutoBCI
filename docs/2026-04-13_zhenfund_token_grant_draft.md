## Q4. 简要介绍你自己

00 后，浙江人，现在北京一家脑机接口研究所的硬件组，是组里唯一的软件工程师——嵌入式、上位机、算法都是我。

我有 ADHD。上课听不进去，考试坐不住，但对自己感兴趣的东西能连续做十几个小时不停。我很早就开始想：为什么同一个大脑，有些事怎么逼自己都做不了，有些事根本不用逼就停不下来？**意识到底是什么？**

这个问题从高中追到现在，一直没换过。

- 高中在课上读自己的书：Minsky《情感机器》、侯世达《GEB》、Gazzaniga《双脑记》、Kurzweil《人工智能的未来》、Nicolelis《脑机穿越》、圣塔菲研究所的复杂系统系列。追的是"意识怎么从神经元里涌现、大脑到底怎么运作"。累计读了 60+ 本，完成率 80%+，不是按课程表读的，是按脑子里的问题读的
- 大一直接进了实验室。导师刚从瑞典皇家理工回国，拿了人才计划在学校开实验室带学生。跟着他从 Git、树莓派、ESP32 一路学起，用实验室 3D 打印机做了 OpenBCI 脑电帽。大二用毫米波雷达做材质识别，拿了华东赛区一等奖（纯手写算法，没有 AI）
- 2022 年 ChatGPT 发布，是我亲历的"iPhone 时刻"——高中读 Minsky 和 GEB 时隐约感到的东西，第一次在真实世界里被验证了
- 然后我去了脑机接口研究所——想亲眼看看生物智能的硬件到底长什么样

接触 AI 辅助编程后学得很快，靠 Claude 和 Codex 从零学了全栈开发和 Agent 工程。

现在在研究所参与了完整的脑机接口链条：猪的开颅手术 → 颅内电极植入 → 脑电采集 → 运动捕捉实验 → 数据集制作 → 比赛出题。中国脑机接口大赛由清华高小榕老师牵头，国内做脑机接口的团队基本都有参与，我是这两年的出题人之一。

曾参加海外 AI Builder Program，demo 获创始人认可并获得资助。2026 年 3 月起全力投入 BCI Agent——不是做更好的脑电算法，而是做一个让 AI Agent 自主进行科学研究的系统。

## Q6. 你看到了什么机会，在做什么产品

**产品名称：AutoResearch**

**机会：**

今天很多 AI 产品在解决“如何帮人写得更快”，但在脑机接口这种实验密集型领域，更大的机会其实是“如何让研究本身变成一个连续运行的系统”。

研究员每天都在重复同一类工作：提假设、改代码、跑实验、看结果、换方向。这个过程很贵、很慢，而且高度依赖少数人的经验。

我自己做过开颅、植入、采集、运动捕捉和数据集制作，越来越强烈地感受到：脑机接口不是单纯缺一个更强的模型，而是缺一个能持续试错、持续换方向、持续生成算法的研究系统。

**产品：**

AutoResearch 是一个让 AI Agent 自主完成科学研究迭代的系统。

它不是 copilot，不是人写代码、AI 补全，而是一个完整的研究闭环：

```
Agent 自己提假设 → 自己改代码 → 自己跑实验 → 自己评估结果
→ 卡住了自己换方向 → 搜索论文找新思路 → 继续迭代
```

我们已经用它跑了数百次自动实验迭代，并开始搭建一套 benchmark，专门测试这种研究框架从零面对新任务时的能力。最近在做的第一条 benchmark，是一个新的步态划分任务：不是沿着既有轨道继续调参，而是看 Agent 能不能从任务定义出发，自己推进研究。

**和现有方案的区别：**

当前 BCI 领域的算法开发，本质上还是手工业：研究员手动选模型、手动调参、手动换方向。我们想做的，是把这个过程变成一个可以连续运行的系统：任务定义清楚后，Agent 能自己提假设、改代码、跑实验、评估结果，并在卡住时继续寻找新方向。

如果这套系统成立，它不只适用于脑机接口。任何需要反复实验、快速试错、不断换方向的科学研究，都可能用同一套框架来推进。

## Q9. 当前 token 消耗和 1 亿 token/天会怎样

**当前消耗与约束：**

当前 AutoResearch 的日均 token 消耗大约在 100-200 万之间。单次 campaign（16 轮自动迭代）约消耗 50-80 万 token。高频 Agent 实验的主要 token 压力目前还是由我个人承担。虽然有过一次性的内部支持，但远不足以覆盖持续实验和订阅成本，所以我现在必须非常克制地分配实验预算。

这意味着我每天通常只能跑 1-2 个 campaign，而且会主动压低搜索和推理深度。现在的瓶颈已经不是“有没有想法”，而是“能不能让系统连续地去试、去比较、去换方向”。

**如果获得 1 亿 token/天：**

变化不会只是“能多调用一点模型”，而是会直接改变系统的运行方式：

1. **从串行试错变成多方向并行推进**：现在很多方向只能一个一个排队尝试，有了更高额度之后，系统才能真正同时探索多个方向，再快速收敛

2. **从节省变成充分搜索和推理**：系统不仅能跑实验，还能在卡住时做更充分的搜索、比较和换向

3. **Benchmark 的迭代速度会明显加快**：现在“改框架 → 跑 benchmark → 分析 → 再改”往往要跨天；token 充足之后，这个循环可以压缩到一天内完成多轮真实对比

4. **从脑机接口走向更一般的实验型研究场景**：脑机接口只是第一站，本质上我们在做的是一个可以被复制到更多研究任务上的 Agent 研究框架

对我来说，token 不是成本项本身，而是这个系统的燃料。它直接决定了 AutoResearch 能不能从“一个人推动的原型”进入“真正连续运转的研究系统”。

## Q10. 做过最酷的事

从一头活猪的颅骨开始。

我参与过完整的脑机接口链条：开颅、植入、采集、运动捕捉、数据集制作，再到比赛赛题设计。这个过程最打动我的，不是某一项技术，而是它让我第一次非常具体地看到：我们对大脑的观测有多有限，信号链路里的偏差有多大。

也正因为这样，我形成了一个和主流直觉不太一样的判断：脑电更像一个宏观状态信号，而不是一个能直接读出精细编码内容的接口。所以我没有继续只做“更好的模型”，而是转向做 AutoResearch，想让 Agent 自己去探索、试错、换方向。

最让我有成就感的，不只是做过开颅手术，而是这条线被我自己接起来了：从高中开始追问“智能是什么”，到后来亲手去看生物智能的硬件，再到今天试着让 AI Agent 自动做科学研究。我觉得这三件事本来是同一个问题的不同阶段。
